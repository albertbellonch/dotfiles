# Setting up

## Dependencies

* Ensure Ruby and brew are installed
* Run `brew install cowsay fortune wget zsh-autosuggestions`
* Run `gem install lolcat`
* Set the `SEQURA_MACHINE` environment variable to `true` if you are on a Sequra machine

## Usage

* Ensure the `zsh/zshrc.symlink` is run, either by creating a symlink to `.zshrc` or doing `source PATH_TO_DOTFILES/zsh/zshrc.symlink`
